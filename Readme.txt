1.DESCRIPTION
1.1 Computer Application game based on electronic game of memory skill invented by Ralph H. Baer and Howard J. Morrison. The application generates a random array of pattern and tracks if the user can remember it all in order to score and move to the higher level.

If the user cannot remember the pattern generated the game will reset and start again.

1.2 Languages and Technologies used
-HTML 5
-CSS 3
-Javascript

2.PURPOSE FOR THIS CODE
To practice programming using Javascript.


3.INSTALLATION
-Download/clone the repository
-Run index.html file via the browser.
